# 🔐 Phishing URL Detection System

This project is a web-based cybersecurity tool that detects potentially malicious URLs using rule-based analysis.
🚀 Features
🔍 Real-time URL analysis
  🟢 Safe / 🟡 Suspicious / 🔴 High-Risk classification
⚠ Detection based on:
    1.Presence of "@" symbol
    2.HTTP (non-secure) usage
    3.Suspicious keywords like login, verify
    4.URL length analysis
🌐 Simple and interactive web interface
